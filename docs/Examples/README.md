# Примеры использования Max Uploader PHP

В этой папке содержатся примеры кода для работы с библиотекой max-uploader-php.

## Список примеров

| Файл                                           | Описание                                                                     |
|------------------------------------------------|------------------------------------------------------------------------------|
| [MaxSimpleUploader.md](./MaxSimpleUploader.md) | Примеры базовой и расширенной загрузки файлов через простой интерфейс        |
| [MaxUploader.md](./MaxUploader.md)             | Примеры продвинутой загрузки с поддержкой потоковой передачи и возобновления |

## Темы примеров

- Базовая загрузка файлов
- Расширенная загрузка с деталями
- Настройка скорости и обработка таймаутов
- Потоковая загрузка больших файлов
